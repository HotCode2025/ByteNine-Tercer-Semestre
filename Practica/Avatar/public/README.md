# Proyecto Avatar
 
## Comandos utilizados para la creación de la estructura del proyecto
 
### 1. Crear la carpeta del proyecto
```bash
mkdir Avatar
```
 
### 2. Ingresar a la carpeta del proyecto
```bash
cd Avatar
```
 
### 3. Crear la carpeta principal `public`
```bash
mkdir public
```
 
### 4. Ingresar a la carpeta `public`
```bash
cd public
```
 
### 5. Crear las subcarpetas
```bash
mkdir assets
mkdir js
mkdir css
```
 
### 6. Crear los archivos base
```bash
touch README.md
touch avatar.html
```
 
### 7. Volver a la carpeta raíz del proyecto
```bash
cd ..
```
 
### 8. Abrir el proyecto en Visual Studio Code
```bash
code .
```
